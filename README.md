# Car Rental Management System

Academic web application developed for a car rental management system using C#, ASP.NET MVC, JavaScript, and SQL Server.

## About the project
This project was developed as part of an academic course to demonstrate practical skills in web development, database management, and application architecture.

The system was designed for a car rental business and includes features for managing customers, vehicles, rentals, and user accounts, as well as email-based communication tools.

## Features
- Customer management
- Vehicle listing and management
- Rental management
- User authentication
- Login, logout, and registration
- Email sending
- Newsletter subscription and email notifications

## Technologies used
- C#
- ASP.NET MVC
- Razor Pages
- JavaScript
- SQL Server
- HTML
- CSS

## Learning outcomes
This project helped strengthen knowledge in:
- MVC architecture
- Object-oriented programming in C#
- SQL Server database design and queries
- Authentication and user management
- Integration of email functionality in web applications

## Purpose
The goal of this project was to build a functional web application for a car rental scenario while applying concepts learned in class, including backend development, frontend integration, and database operations.
